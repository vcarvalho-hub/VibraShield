Aqui está a descrição dos componentes fisícos utilizados para a construção do projeto
# Hardware

## Materiais utilizados

### ESP32
Microcontrolador responsável pelo processamento dos dados e comunicação do sistema.

### Sensor Piezoelétrico
Utilizado para detectar vibrações mecânicas na estrutura monitorada.

### Sensor de Som
Responsável por captar variações sonoras durante os testes.

### LEDs
Indicadores visuais do estado do sistema.

### Protoboard
Utilizada para montagem do circuito sem necessidade de soldagem.

### Jumpers
Cabos utilizados para interligar todos os componentes.
